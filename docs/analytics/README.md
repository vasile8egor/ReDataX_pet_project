# Analytics

Analytics connects experiment artifacts to ClickHouse tables, Metabase dashboards, and research reports.

Existing baseline exports are stored under `docs/analytics/baseline/`.

## Responsibilities

- Define metrics consistently.
- Load experiment artifacts into analytical tables.
- Maintain dashboard naming and filters.
- Produce reproducible reporting snapshots.
