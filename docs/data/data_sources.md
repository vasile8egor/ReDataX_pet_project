# Data Sources

## Synthetic Sources

Synthetic sources are generated by project code for accounts, transactions, FX requests, market rates, and inventory states.

## Real-Market Sources

The primary real-market source is Binance public spot `aggTrades` archives.

Canonical symbols:

- `BTCUSDT`
- `ETHUSDT`
- `ETHBTC`

## Storage

Raw downloads should be preserved before transformation. Parsed and feature-ready forms are loaded into analytical storage for experiments.
